мы используем

В этом блоке показаны технологии которые мы использовали при разработке наших проектов. Так же мы оставим ссылки на дополнительный материал для более подробного ознакомления.

HTML

Стандартизированный язык разметки документов для просмотра веб-страниц в браузере. Проще говоря с помощью него мы создаем элементы на странице

CSS

Формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Проще говоря с помощью него мы меняем созданные HTML элементы

JavaScript

Мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Проще говоря с помощью него мы наделяем функционалом наши элементы

GSAP

Это простая в использовании библиотека JavaScript для веб-анимации. Она позволяет анимировать практически все, к чему можно получить доступ с помощью JavaScript, включая SVG, общие объекты, холсты и многое другое.
